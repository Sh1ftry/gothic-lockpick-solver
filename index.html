HOW TO USE

Number of gates — use the + and − buttons to add or remove gates (min 4, max 7).

Set gate positions — drag any gate left or right to position it.

Program links — click a gate to select it (highlighted in blue). Every other gate shows two tap zones:

  O (left edge) — that gate moves in the opposite direction when the selected gate is moved.
  S (right edge) — that gate moves in the same direction as the selected gate.

Tap O or S to toggle.

Solve — once positions and links are set, press SOLVE to get the shortest sequence of moves that centres every gate.

Restart — hold the restart button for 2 seconds to reset all gates and links back to the default state.
